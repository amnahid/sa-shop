# Settings & Administration

Configure and manage your entire business structure and system preferences from a centralized settings hub.

## Features

### Business Profile
- **Legal Entity Info**: Manage your business name (EN/AR), address, and contact details.
- **Tax Configuration**: VAT registration number (15-digit), registration status, and ZATCA phase status.
- **Branding**: Upload your business logo to be displayed on dashboard and receipts.
- **Localization**: Set base currency (SAR), timezone, and default language.

### Branch Management
- **Location Mapping**: Define physical branches with unique names, addresses, and phone numbers.
- **Head Office**: Designate a main branch for primary inventory and financial aggregation.
- **VAT Branch Codes**: Configure specific codes required for ZATCA multi-branch reporting.

### System Preferences
- **POS Defaults**: Set default VAT rates, payment methods, and receipt delivery preferences.
- **Security**: Enable MFA (Multi-Factor Authentication) and view system access logs.
- **Integrations**: Manage connections to external services (Email, WhatsApp, SMS).

## Compliance
- **ZATCA**: All tax-related setting changes are logged in the audit trail.
- **PDPL**: Manage data retention policies and privacy consent versions.

## User Roles
- **Owners**: Full access to all settings and configuration.
- **Managers**: Limited access to branch and profile settings; no access to tax or security configuration.
