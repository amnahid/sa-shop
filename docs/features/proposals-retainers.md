# Proposals & Retainers

The system supports business-to-business (B2B) workflows through professional quotes and recurring service agreements.

## Features

### Proposals (Quotes)
- **Drafting**: Create detailed quotes with multiple items, quantities, and custom descriptions.
- **Expiry Dates**: Set validity periods for proposals.
- **Status Tracking**: Track proposals through stages: Draft, Sent, Accepted, Rejected.
- **Convert to Invoice**: Instantly convert an accepted proposal into a standard tax invoice.
- **PDF Generation**: Professional, branded PDF generation for sharing with clients via email or WhatsApp.

### Retainers (Recurring Services)
- **Service Agreements**: Manage ongoing service contracts with recurring billing.
- **Billing Cycles**: Support for monthly, quarterly, or annual billing.
- **Automated Reminders**: Notifications when a retainer is up for renewal or billing.

## Target Use Cases
- **B2B Bulk Sales**: Providing quotes to schools, offices, or other businesses for bulk grocery or cleaning supplies.
- **Corporate Services**: Ongoing supply arrangements for companies with monthly settlements.

## User Roles
- **Managers**: Can create and manage all proposals and retainers.
- **Owners**: Full visibility and approval authority.
